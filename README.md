# Lura 

A document viewer written in Python with PyQt.

## Install

```console
$ git clone https://github.com/aotabekov91/lura
$ cd lura
$ poetry install
```

## Usage

```console
$ poetry run lura /path/to/file.pdf
```

At the initial run: 

* Enter the exec mode (by default <c-e>)
* Select installPicks 

This will install plug dependencies.
