# Lura 

A document viewer written in Python with PyQt.

## Install

```console
$ git clone https://github.com/aotabekov91/lura
$ cd lura
$ poetry install
```

After the initial install, enter the exec mode (press \<c-.\>) and type installPicks and enter to enter plugins. Then rerun the app.
To exit press \<c-q\>.

## Usage

```console
$ poetry run lura /path/to/file.pdf
```
